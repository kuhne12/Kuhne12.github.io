机器学习作业chp7

本次练习的第一部分将实现异常检测算法，并将其应用于检测网络服务器故障。第二部分将使用协同过滤来建立一个电影推荐系统。

一、异常检测

这部分将实现异常检测算法来检测服务器的异常行为。通过输入（mb/s）和延迟（ms）两个特征，衡量每个服务器的响应速度，在服务器运行时，收集了 m=307m=307m=307 个服务器行为样本，得到了一个无标签数据集 x1,⋅⋅⋅,xm{x^1,···,x^m}x1,⋅⋅⋅,xm ，我们认为数据集中大多数都是正常（非异常）的样本，但是也可能有一些异常的样本。本次实验将使用高斯模型来检测数据集中的异常样本，首先需要将数据集可视化，以便理解算法的工作原理。在数据集中将先拟合高斯分布，然后找到概率特别低的值，将其视为异常样本。最后将把异常检测算法应用于一个多维的大数据集中。

协同过滤的代价函数（不带正则项）是：

强烈建议使用向量化的方式实现损失函数 JJJ 的计算，因为它之后将会被高级优化函数调用很多次。通常情况下，最简单的方式是先写出一个非向量化的实现（保证你取得了正确的答案），随后修改代码，使其实现向量化（注意向量化后的输出不能改变）。为了得到向量化的实现，有一个很有效的建议：可以使用 RRR 矩阵去将选定的条目设置为 0。比如 R.*M 将在矩阵 RRR 和 MMM 之间做一个元素级乘法运算，因为 RRR 矩阵中的元素只有 0 或 1，这样做的效果是，只有当 RRR 中相应的值为 0 时，才会将 MMM 的元素设置为 0。
