三、数据预处理

label 排序

我们拥有的数据是杂乱的标签，为了能够进行训练，我们需要将这些混乱的标签进行标号。第一步需要将所有混乱的 label 进行一定的排序。

label 转换

实现了 label 排序后，我们需要将真正的 label 转换为对应的数字 label，同时，由于比赛提交的文件是图像的真正 label ，我们还需要实现一个将数字 label 转化为真正label 的函数

定义 Dataset

我们通常会使用 Dataset + DataLoader 组合的方式来获取迭代数据，因为使用的是自己的数据集，所以要重写一下 Dataset 类。

重写 Dataset 类需要注意重写三个函数：

  • __init__()：用于初始化 self 的数据，这个方法里面定义的参数就使调用该类所需要填写的参数，包括：
    • data_path：数据集存放的路径
    • image_path：图像存放的路径
    • mode：运行模式，分为：
      • train
      • val
      • test
    • val_ratio：验证集占数据集的比例
  • __getitem__()：这个方法是用于索引得到数据的
  • __len__()：返回数据的长度

因为有三种模式，且三种模式所需要的数据都不相同，所以要根据这三种情况分别获取数据

定义 DataLoader

有了 Dataset 后，就可以定义 Data Loader 了，分别有 train_dataloder，val_dataloader，test_dataloader
