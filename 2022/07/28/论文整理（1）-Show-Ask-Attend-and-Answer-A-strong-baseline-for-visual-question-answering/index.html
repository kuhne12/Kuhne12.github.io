论文整理（1） Show, Ask, Attend, and Answer-A strong baseline for visual question answering

发表于2022-07-28|更新于2023-03-04|深度学习
|字数总计:2.9k|阅读时长:10分钟|阅读量:

摘要

这篇 paper 是视觉问答的一个 baseline，给出图像和用自然语言描述的问题，模型可以根据图片内容准确地给出答案。

作为一篇 baseline，模型的结构较为简单，参数也相对较少，但是效果不错：

  • 在 VQA1.0 数据集上，模型在测试集达到了 64.6% 的精确度，且没有使用额外的数据
  • 在 VQA2.0 数据集上，模型在验证集达到了 59.7% 的精确度，比之前报道的最好的结果搞了 0.5%

一、介绍

虽然深度学习在 CV 和 NLP 领域取得了巨大的成就，但是对图片理解而言，计算机与人类的理解水平仍然相差甚远。因为人类在观察图像的同时，还会联想图片内在的状态和属性，而视觉问答领域的研究，可以让我们理解模型在图像中真正看到了什么。

与视觉问答相关的一个研究领域是 image caption generation（图像标注生成），即：根据输入的图像，生成一段用于描述该图像内容的句子，是模型理解图像的另一种角度。但是两者也有一定的区别，VQA 是针对提出的问题来回答明确的答案，而图像标注生成的语言没有针对性，没有唯一的用来标注一幅图的句子。而且，用一句模糊的语言来描述一幅图片是很容易的，但这种表示并不能代表模型理解了图像。而 VQA 则不同，模型必须准确理解图像内容，得到的回答要有针对性。

这篇 paper 中使用的模型虽然简单，但也能打败那个时候最前沿的模型。下面是本模型的结构图：

从结构上来看，它们提出的模型采用了：

  • 长短时记忆网络（LSTM）编码问题；

  • 深度残差网络（ResNet）来计算图像特征；

  • 连接图像特征和 LSTM 的最终状态，利用软注意力机制计算图像特征 glimpses（每一个 glimpse 对应一个物体的part）

  • 随后分类器将 glimpses 和 LSTM 的最终状态作为输入，在固定的答案集上生成对应的可能性

二、相关工作

  • 卷积神经网络（CNN）

    • 激活函数的选择
    • 权值初始化
    • 优化器
    • 正则化

  • 注意力机制（Attention）

    • 使深度神经网络能够从输入中提取局部特征

  • 长短时记忆网络（LSTM）

    • 在机器翻译和语音识别方面有很杰出的表现

  • SANs（Stacked Attention Networks for Image Question Answering）

    • 本文算法与 （SANs）一样，只有一些细微的差别。
    • 在 VAQ1.0 数据集上的表现有 5.8% 的提升

三、方法

本文将视觉问答任务视为一个分类问题，定义：给定一张图像 III，和用自然语言描述的问题 qqq，基于图像内容，需要在固定的答案集中，估计最有可能的答案 a^\hat{a}a^ 。数学公式定义：

a^=argmaxP(a∣I,q)\hat{a}=argmaxP(a|I,q) a^=argmaxP(a∣I,q)

其中 a∈{a1,a2,⋅⋅⋅,aM}a\in\{a_1,a_2,···,a_M\}a∈{a1​,a2​,⋅⋅⋅,aM​} .

3.1 Image embedding（图像嵌入）

本模型使用预训练的 ResNet ，计算输入图像 III 的高级特征 ϕ\phiϕ .

ϕ=CNN(I)\phi=CNN(I) ϕ=CNN(I)

这里对 ϕ\phiϕ 的理解是：一共有 14×14=19614\times14=19614×14=196个位置，每个位置都用 2048 维的特征向量来表示，196 个位置放大到原图，大概是 21×2121\times2121×21 大小的图像像素空间。

3.2 Question embedding（问题嵌入）

将问题使用词嵌入（Word Embedding）编码成为一个长度为 PPP 的序列（PPP 是问题的长度）

3.3 Stacked attention（堆叠注意力）

类似于 SANs，本模型在图像特征的空间维度，计算了多重注意力分布。

3.4 分类

最后，我们将图像 glimpses 和 LSTM 的最终状态连在一起，使用非线性变换计算每个答案的可能性

四、 实验

4.1 数据集

4.1.1 VQA 1.0

VQA 1.0 数据集由来自于 MS COCO 数据集的 204, 721 张照片构成，并使用开源的 614, 163 个问题和 6, 141, 630 个答案评估本模型。

4.1.2 VQA 2.0

同时作者也使用了由 658, 111 个问题和 6, 581, 110 个答案构成的 VQA 2.0 来评估模型，相对于 VQA 1.0 ，该版本的数据集更加平衡，每个问题对应两张图像，两张图像对同一个问题会有两个不同的答案。

五、结果

5.1 Baseline

所有输入图像在保留纵横比和中心的同时，被缩放至 299×299299\times299299×299 （作者发现拉伸图像不利于模型的表现）。

图像特征提取于与训练好的 152 层 ResNet，在平均池化层前取出尺寸为 14×14×204814\times14\times204814×14×2048 的张量，并采用 l2 正则化。

输入问题被嵌入为 D=300D=300D=300 维的向量序列，该序列在喂入 LSTM 之前首先经过 tanh 非线性函数，为了适用于不同长度的问题，采用了动态展开。

为了在图像特征上计算注意力，作者将 LSTM 的最终状态和图像特征连接在一起，再经过一个深度为 512 的 1×11\times11×1 的卷积层，随后接着一个 ReLU 非线性激活函数。输出特征被传入另一个深度为 2 的 1×11\times11×1 的卷积层，随后跟着一个 Softmax ，从而计算出注意力分布。

接着使用注意力分布，通过加权平均的图像特征计算两个图像 glimpses 。

随后作者进一步连接了图像 glimpses 和LSTM 的最终状态，并将链接后的结果传入包含 1024 个 ReLU 非线性激活函数的全连接层，最后产生的结果被喂入大小为 3000 的线性层（仅考虑前 3000 个最频繁的答案。），用于生成每一个答案的可能性。

其他的一些设置有：

  • 用 0.5 的 Dropout 处理输入特征（包括 LSTM、CNN、FC）

  • 使用 Adam 优化器，batch size 为 128

  • 初始学习率 l0=0.001l_0=0.001l0​=0.001 ，并使用指数学习率衰减，公式为

    lstep=0.5stepdecay stepl0l_{step}=0.5^{\frac{step}{decay\ step}}l_0 lstep​=0.5decay stepstep​l0​

    衰减步数为 50K，β1=0.9\beta_1=0.9β1​=0.9，β2=0.999\beta_2=0.999β2​=0.999

  • 训练期间 CNN 的参数保持不变

默认模型及其变种在不同训练步上的精确度如下图所示：

对于测试的结果，作者总结如下：

  • 采用 l2 正则化的模型可以提高模型的泛化能力，在减少训练时间的同时也能很好地提高模型精度

  • 在不同层采用 dropout 对避免过拟合是至关重要的

  • 使用软注意力能显著提高模型的精确度

  • 不同的文本嵌入大小和 LSTM 的配置不是影响模型精度的主要因素，使用较大的嵌入尺寸搭配较小的 LSTM 似乎是最佳选择；

  • 使用了损失采样似乎会显著降低模型的性能，且延长训练时间

  • 不同于 SANs 的报告，作者发现使用堆叠注意力机制只能略微改善结果

  • 使用两层分类器的效果比使用单层分类器好得多，而增加或减少分类器的宽度似乎不会提高模型的性能

六、总结

这篇 Paper 展现了 VQA 的一个新的 Baseline，尽管结构简单，且本质上与之前的模型很相似，该模型在 VQA 1.0 和 VQA 2.0 数据集上优于之前报道的结果。

该模型做的也是一个分类任务，因为最后的输出还是在可选范围内的，文中使用了3000个类别，只是需要根据问题针对图中的情况进行选择一个类别
