一、深度学习框架

1.1 深度学习框架介绍

在进行深度学习时，都需要考虑构建一个计算图，来表示任何想要计算的函数，在线性分类器的情况下，我们会对输入数据 XXX 和权重矩阵 WWW 进行矩阵乘法，可能会使用合页损失函数计算损失值，用正则项防止模型过拟合。大型神经网络的图结构会非常复杂，包括很多不同的层，很多不同的激活函数，很多不同的权重。

如果将目光转到神经图灵机，就可能得到下方这种非常“疯狂”的计算图，有时甚至不能绘制出来。

所以深度学习框架意义重大，有三个使用深度学习框架，而不是自己编写代码的主要原因：

  1. 非常轻松地构建和使用一个庞大的计算图，无需在意细节；
  2. 非常轻松地在计算图中计算梯度；
  3. 将所有地运算都高效地运行在 GPU 上。

二、PyTorch

2.1 PyTorch 基本使用

PyTorch 是 Facebook 发行的深度学习框架，内部由三个部分组成：

  • Tensor （张量）

    PyTorch 中的张量就像 Numpy 中的数组，但是他可以在 GPU 中运行，是一种最基本的数组。

  • Autograd（变量）、

    Autograd 是 PyTorch 中的一个包，用于建立计算图和反向传播，以达到自动计算梯度的功能。

  • Module（模对象）

    模对象是一个神经网络层，我们可以将这些模组合起来建立一个大的网络，并且可以存储一些状态值或可学习的权重。

三、静态图 VS. 动态图

3.1 动态图
