线性回归、代价函数、梯度下降 2022/4/14

一、线性回归

问题描述：有一个住房价格的数据集，我们希望从这个数据中学习预测房价。

在这个过程中，需要定义许多符号：

  • mmm：训练样本的数量

  • xxx：输入变量

  • yyy：输出变量

  • (x,y)(x,y)(x,y)：一个训练样本

  • (x(i),y(i))(x^{(i)},y^{(i)})(x(i),y(i))：表示第 iii 个样本 (方便起见，有时候右上角的括号会被省略)

监督学习算法工作方式为：提供训练集 →\rightarrow→ 机器学习算法 →\rightarrow→ 输出目标函数

假设函数表达式如下：

hθ(x)=θ0+θ1∗xh_ \theta(x) = \theta_0+\theta_1 * x hθ​(x)=θ0​+θ1​∗x

在图中画出函数，意味着我们预测 yyy 是关于 xxx 的线性函数，这就是数据集：

这个例子是一个 一元线性回归 ，又称为 单变量线性回归 ，变量是x ，通过这个模型可以预测所有的价格。

（由于线性模型是学习的基础，所以先从这个例子开始，拟合线性函数，之后在此基础上处理更复杂的模型。）

二、代价函数(Cost Function)

代价函数是为了帮助我们找到最适合的参数，从而用最适合的直线拟合数据

2.1 代价函数的数学定义

上一节中介绍的假设函数

hθ(x)=θ0+θ1∗xh_ \theta(x) = \theta_0+\theta_1 * x hθ​(x)=θ0​+θ1​∗x

中，θ0\theta_0θ0​ 和 θ1\theta_1θ1​ 被称为模型参数，选择不同的参数，我们会得到不同的假设函数( hθ(x)h_\theta(x)hθ​(x) 函数)，以下为不同参数取值得到的图像：

在线性回归问题中，我们需要做的是得出 θ0\theta_0θ0​ 和 θ1\theta_1θ1​ 这两个参数，让假设函数与数据点能够很好的拟合。

那该如何找出这两个参数呢？

我们的想法是，要选择能使 h(x)h(x)h(x) 的结果最接近该样本 xxx 对应的 yyy 值的参数 θ0\theta_0θ0​ 和 θ1\theta_1θ1​ 。即：找到 (h(x)−y)2(h(x)-y)^2(h(x)−y)2 的最小值，而 (xi,yi)(x^i,y^i)(xi,yi) 表示第 iii 个样本，所以我们要做的是对所有训练样本进行求和，计算式为 ∑i=1m(h(xi)−yi)2\sum_{i=1}^{m}(h(x^i)-y^i)^2∑i=1m​(h(xi)−yi)2 ，为了使数据更加直白一些，我们取这个和的平均值的 12\frac{1}{2}21​ ，最终得到计算式为 12m∑i=1m(h(xi)−yi)2\frac{1}{2m}\sum_{i=1}^{m}(h(x^i)-y^i)^22m1​∑i=1m​(h(xi)−yi)2 ，这将是线性回归的整体优化目标函数。

为了使这个函数更加明确，我们改写这个函数，定义一个代价函数：

J(θ0,θ1)=12m∑i=1m(h(xi)−yi)2J(\theta_0,\theta_1)=\frac{1}{2m}\sum_{i=1}^{m}(h(x^i)-y^i)^2 J(θ0​,θ1​)=2m1​i=1∑m​(h(xi)−yi)2

代价函数也被称为平方误差函数，有时也被称为平方误差代价函数，除了它以外，别的代价函数也能很好地发挥作用，但是平方误差代价函数是解决回归问题最常用的手段。

2.2 代价函数的作用及使用原因

  • 仅有一个参数的代价函数

    为了使代价函数 JJJ 可视化，我们使用一个简化的代价函数，令 θ0=0\theta_0=0θ0​=0 ，使他仅包含一个参数 θ1\theta_1θ1​ ，代价函数变为 J(θ1)J(\theta_1)J(θ1​) ，相当于只选择了经过原点的假设函数。

    假定数据集为 (1,1),(2,2),(3,3)(1,1), (2,2), (3,3)(1,1),(2,2),(3,3) ，我们取 θ1=1\theta_1=1θ1​=1，经过上述代价函数数学表达式，得出 J(θ1)J(\theta_1)J(θ1​) 的值刚好为 0。随后我们取不同的 θ1\theta_1θ1​ 值，绘制出 J(θ1)J(\theta_1)J(θ1​) 的函数图像：

    而机器学习算法的优化目标，就是通过选择 θ1\theta_1θ1​ 的值，获得最小的 J(θ1)J(\theta_1)J(θ1​)，可以看出，当 θ1=1\theta_1=1θ1​=1 时，J(θ1)J(\theta_1)J(θ1​) 最小，而通过绘图可以知道，这确实是一条符合数据的直线。

  • 有两个参数的代价函数

    当参数增加为两个时，代价函数如下图所示，曲面高度即为 J(θ0,θ1)J(\theta_0,\theta_1)J(θ0​,θ1​) 的值。

    该函数等高线表示如下，图中每一条线都代表相同的值：

    取一个特定的点，找到对应的 θ1\theta_1θ1​ 和 θ0\theta_0θ0​ ，就可以绘制出对应的假设函数， 根据代价函数最小的值对应的 θ1\theta_1θ1​ 和 θ0\theta_0θ0​ 画出的图像，是最接近数据集的直线。

三、梯度下降

梯度下降是一种可以将代价函数 J(θ0，θ1)J(\theta_0，\theta_1)J(θ0​，θ1​) 最小化的算法，它不仅能被用在线性回归问题上，还能应用于机器学习的众多领域。

  • 给 θ1\theta_1θ1​ 和 θ0\theta_0θ0​ 赋上初值，一般都设置为 0；

3.2 梯度下降数学算法定义

数学定义：

θj:=θj−α∂∂θjJ(θ0,θ1)\theta_j := \theta_j-\alpha \frac{\partial}{\partial \theta_j}J(\theta_0,\theta_1) θj​:=θj​−α∂θj​∂​J(θ0​,θ1​)

同时，θ0\theta_0θ0​ 与 θ1\theta_1θ1​ 的值也要 同步 更新（顺序不能变）：

temp0:=θ0−α∂∂θ0J(θ0,θ1)temp_0 := \theta_0-\alpha \frac{\partial}{\partial \theta_0}J(\theta_0,\theta_1) temp0​:=θ0​−α∂θ0​∂​J(θ0​,θ1​)

temp1:=θ1−α∂∂θ1J(θ0,θ1)temp_1 := \theta_1-\alpha \frac{\partial}{\partial \theta_1}J(\theta_0,\theta_1) temp1​:=θ1​−α∂θ1​∂​J(θ0​,θ1​)

θ0:=temp0\theta_0 := temp_0 θ0​:=temp0​

θ1:=temp1\theta_1 := temp_1 θ1​:=temp1​

其中特殊的符号意义：

  • := 表示赋值，将右侧的值赋给左侧
  • ∂∂θjJ(θ0,θ1)\frac{\partial}{\partial \theta_j}J(\theta_0,\theta_1)∂θj​∂​J(θ0​,θ1​)：代价函数对参数 θ\thetaθ 的导数，以下详细讨论

根据之前的学习，绘制出 J(θ1)J(\theta_1)J(θ1​) 的函数图像：

随后从 θ1\theta_1θ1​ 这个点开始梯度下降，不断更新。 θ1:=θ1−αddθ1J(θ1)\theta_1 := \theta_1-\alpha \frac{d}{d\theta_1}J(\theta_1)θ1​:=θ1​−αdθ1​d​J(θ1​) ，由于 α\alphaα 是一个正数，J(θ1)J(\theta_1)J(θ1​) 的导数也是正数，可以知道 θ1\theta_1θ1​ 的值一直在减小，逐渐接近最低点对应的值。

从左侧看同理，θ1\theta_1θ1​的值会增大，更加接近最低点对应的值。

  • α\alphaα ：表示学习率，是一个实数，用来控制梯度下降的速度

    • 如果α\alphaα太小，梯度下降的速度会很慢

    • 如果α\alphaα太大，那么梯度下降可能会越过最低点，甚至无法收敛甚至会发散

假如 θ1\theta_1θ1​ 初始化在局部最低点，下一步该怎么进行梯度下降呢？

当处于局部最低点时，J(θ1)J(\theta_1)J(θ1​)的导数值为 0，根据公式，θ1\theta_1θ1​ 的值将不会改变，使解一直保持在局部最优点。同时，当我们不断靠近最低点时，可以发现 J(θ1)J(\theta_1)J(θ1​)的导数会越来越小，梯度下降会自动采取较小的步幅，所以实际上没必要再另外减小 α\alphaα 。

3.3 线性回归的梯度下降

通过以上的学习，我们知道了梯度下降算法

θj:=θj−α∂∂θjJ(θ0,θ1)\theta_j := \theta_j-\alpha \frac{\partial}{\partial \theta_j}J(\theta_0,\theta_1) θj​:=θj​−α∂θj​∂​J(θ0​,θ1​)

以及线性回归模型，

hθ(x)=θ0+θ1∗xh_ \theta(x) = \theta_0+\theta_1 * x hθ​(x)=θ0​+θ1​∗x

J(θ0,θ1)=12m∑i=1m(h(xi)−yi)2J(\theta_0,\theta_1)=\frac{1}{2m}\sum_{i=1}^{m}(h(x^i)-y^i)^2 J(θ0​,θ1​)=2m1​i=1∑m​(h(xi)−yi)2

接下来我们要将梯度下降算法应用到最小化平方差代价函数。 为了应用梯度下降算法，关键步骤就是处理导数项∂∂θjJ(θ0,θ1)\frac{\partial}{\partial \theta_j}J(\theta_0,\theta_1)∂θj​∂​J(θ0​,θ1​)，将该导数项化简后得到：

∂∂θjJ(θ0,θ1)=∂∂θj⋅12m⋅∑i=1m(θ0+θ1xi−yi)2\frac{\partial}{\partial \theta_j}J(\theta_0,\theta_1)=\frac{\partial}{\partial \theta_j}·\frac{1}{2m}·\sum_{i=1}^{m}(\theta_0+\theta_1x^i-y^i)^2 ∂θj​∂​J(θ0​,θ1​)=∂θj​∂​⋅2m1​⋅i=1∑m​(θ0​+θ1​xi−yi)2

分别对 θ0和θ1\theta_0 和 \theta_1θ0​和θ1​ 求导，得到

∂∂θ0J(θ0,θ1)=1m⋅∑i=1m(h(xi)−yi)\frac{\partial}{\partial \theta_0}J(\theta_0,\theta_1)=\frac{1}{m}·\sum_{i=1}^{m}(h(x^i)-y^i) ∂θ0​∂​J(θ0​,θ1​)=m1​⋅i=1∑m​(h(xi)−yi)

∂∂θ1J(θ0,θ1)=1m⋅∑i=1m(h(xi)−yi)⋅xi\frac{\partial}{\partial \theta_1}J(\theta_0,\theta_1)=\frac{1}{m}·\sum_{i=1}^{m}(h(x^i)-y^i)·x^i ∂θ1​∂​J(θ0​,θ1​)=m1​⋅i=1∑m​(h(xi)−yi)⋅xi

得到

θ0:=θ0−α⋅1m⋅∑i=1m(h(xi)−yi)\theta_0:=\theta_0-\alpha·\frac{1}{m}·\sum_{i=1}^{m}(h(x^i)-y^i) θ0​:=θ0​−α⋅m1​⋅i=1∑m​(h(xi)−yi)

θ1:=θ1−α⋅1m⋅∑i=1m(h(xi)−yi)⋅xi\theta_1:=\theta_1-\alpha·\frac{1}{m}·\sum_{i=1}^{m}(h(x^i)-y^i)·x^i θ1​:=θ1​−α⋅m1​⋅i=1∑m​(h(xi)−yi)⋅xi

从前面知道，梯度下降很容易陷入局部最优解（到达局部的最小值后不再下降），但是线性回归的代价函数总是一个凸函数（这里的凸函数和数学中的凸函数是不一样的）， 只有一个全局最优解。

画出假设函数 h(x)h(x)h(x) 与代价函数 J(θ0,θ1)J(\theta_0,\theta_1)J(θ0​,θ1​) 的图像， 先取 θ0=900,θ1=−0.1\theta_0=900,\theta_1=-0.1θ0​=900,θ1​=−0.1 ，随后我们不断梯度下降：

可以发现，函数越来越符合样本集，直到到达全局最小值，最终得到的直线就可以用来预测房价。

以上使用的梯度下降有时被称作Batch梯度下降，每一步梯度下降都要遍历整个训练集（因为计算了总和）。
除了梯度下降，机器学习中还有其他的优化方式，每次只关注小子集，称为正规方程组方法，但是梯度下降适合更大的数据集。
